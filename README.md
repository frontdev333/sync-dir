# sync-dir

`sync-dir` — это CLI-утилита для однонаправленной синхронизации каталогов по содержимому файлов (через SHA-256). Она помогает быстро привести папку назначения к состоянию папки источника: скопировать новые файлы, обновить измененные и удалить лишние.

## Установка и сборка

1. Проверьте версию Go (в проекте указана `go 1.25.1`):

```bash
go version
```

2. Перейдите в директорию проекта:

```bash
cd sync-dir
```

3. Установите зависимости:

```bash
go mod download
```

4. Соберите бинарник:

```bash
go build -o sync-dir .

```

## Примеры использования

> Формат запуска: `sync-dir <source_path> <destination_path>`

### 1) Запуск через `go run`

```bash
go run . ./test-data ./test-data2
```

Что произойдет:
- файлы, которых нет в `./test-data2`, будут скопированы из `./test-data`;
- файлы с отличающимся содержимым будут обновлены;
- файлы, которых нет в источнике, будут удалены из папки назначения.

### 2) Запуск собранного бинарника

```bash
./sync-dir ./test-data ./test-data2
```

### 3) Обратная синхронизация (поменять направление)

```bash
./sync-dir ./test-data2 ./test-data
```

### 4) Синхронизация произвольных директорий

```bash
./sync-dir /path/to/source /path/to/destination
```
